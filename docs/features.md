# features

- a map rendered as static HTML on GitLab Pages
- an unofficial / work-in-progress caution banner pinned above the header
- a sidebar "data freshness" panel: the UTC build time and a live UTC clock
- showing all drifter locations
- toggle for showing trajectories of selected drifters
- toggle for a per-drifter current-advection forecast (dashed line to +6 h, dots at 1/3/6 h)
- toggle for a per-drifter current-advection hindcast (dashed line back to −6 h, dots at 1/3/6 h)
- later (when deployed in batches) allow for selecting batches of drifters
- overlay of today's cmems surface currents (analysis / forecast t=0)
